# Папка для скриптов

Оптимизированная загрузка скрипта

```js
    <script src="scripts/index.js" defer></script>
```

Простые сценарии можно написать в `index.js`, а также есть возможность создать модульную систему для несколько сценариев:

```shell
└── source/
    └── scripts/
        ├── index.js
        ├── modal.js
        └── nav.js
```
